One of my favorite analytics projects explored Airbnb listing data to identify pricing trends and demand patterns.

**GitHub Repository:** [airbnb-listing-insights](https://github.com/agaffney-hub/airbnb-listing-insights)

## Project Overview

This project analyzes Airbnb listing data to uncover the factors that influence pricing, guest satisfaction, and listing performance. Through exploratory data analysis and visualization, I examined relationships between property characteristics, review scores, and nightly rates.

## Key Questions

- What features are associated with higher-priced listings?
- Which factors correlate with stronger guest ratings?
- How do location and property attributes affect demand?

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Highlights

- Performed data cleaning and exploratory analysis on Airbnb listing data.
- Built visualizations to identify pricing and review trends.
- Investigated characteristics common among highly rated properties.
- Generated actionable insights for hosts and market analysis.
